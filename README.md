## Feature

- Asynchronous
- ORM based
- Galleryinfo, Info auto mirroring
- pixiv.net, hitomi.la image proxying
- Search by title

## Deployment environment

- Ubuntu 20.04.1 LTS
- Mongodb atlas (Using Atlas Search)
- Mariadb 15.1 (Self Hosting)
- Nginx (Reverse Proxing)
- Docker
